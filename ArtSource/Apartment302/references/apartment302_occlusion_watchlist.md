# Apartment 302 Occlusion Watchlist

Use this note during room-dressing or screenshot review when clutter starts to threaten clue readability.

This is a laptop 2 guardrail for visual blocking only. It does not change clue logic, prompts, or interaction range.

## Goal

Protect the evidence path from entry to the correct ritual anchor as more domestic clutter, dressing, and false leads are added.

## High-risk occlusion zones

### 1. Entry to nursery cluster

Protect:

- `A302_Clue_CradleHalo`
- `A302_Clue_CardCrown`
- `A302_Clue_BabyMonitor`

Do not let these block the first read:

- tall furniture edges
- hanging cloth shapes
- dark foreground props near the entry frame

### 2. Nursery cluster to answer line

Protect:

- `A302_Anchor_NurserySigil`
- the shared halo / chalk / ash language between the clue cluster and the sigil

Do not let these break the answer read:

- floor clutter between the cluster and the sigil
- oversized rug breakup
- bright dressing props that sit between the clues and the answer

### 3. False-lead spill risk

Watch:

- `A302_Trap_MirrorCircle`
- `A302_FalseLead_WindowLatch`
- `A302_FalseLead_PawnPouch`

Do not let these turn into one combined distraction wall:

- stacked props near the mirror
- a strong line of sight from entry straight into the mirror trap
- window-side clutter that reads louder than the nursery cluster

## Fast review questions

When checking screenshots or a placement pass, answer these quickly:

1. Does entry still read nursery first?
2. Is the correct sigil still readable from the clue family without a second sweep?
3. Did a new prop block the halo silhouette or the chalk read?
4. Did the mirror trap gain a cleaner or brighter sightline than the true answer?
5. Did the window and pawn false leads collapse into one noisy corner?

## Safe fixes inside this lane

- move or lower clutter that blocks the nursery clue family
- reduce contrast on props that sit between the clue cluster and the correct sigil
- break up false-lead clustering so the room pulls attention in different directions
- keep the mirror trap visible without giving it the clearest route from entry

## Out of scope

- changing clue ownership
- changing prompt text
- changing ritual logic
- changing interaction range
