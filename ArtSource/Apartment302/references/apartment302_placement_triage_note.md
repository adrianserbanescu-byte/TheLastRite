# Apartment 302 Placement Triage Note

Use this note after a screenshot review when the issue is clear but the next in-scope room-dressing adjustment still needs to be chosen quickly.

Pair it with:

- `apartment302_screenshot_checklist.md`
- `apartment302_false_lead_balance_guide.md`
- `apartment302_occlusion_watchlist.md`

## Goal

Map the most common screenshot failures to the smallest laptop 2 readability fix that can be tried before escalating or broadening scope.

## Triage table

| Screenshot failure | Likely cause inside this lane | Smallest next fix to try |
| --- | --- | --- |
| `A302_Shot_01_EntryWide` reads mirror-first | mirror trap has the cleanest silhouette or brightest contrast | lower mirror-adjacent contrast or break the door-to-mirror sightline with softer clutter |
| `A302_Shot_01_EntryWide` reads window-first | window frame or curtain treatment is out-pulling the nursery cluster | reduce window emphasis and restore a stronger crib-side frame |
| `A302_Shot_03_NurseryAnchor` feels detached | clutter or prop spacing broke the clue-to-answer line | clear the visual route from nursery clue family to `A302_Anchor_NurserySigil` |
| `A302_Shot_04_MirrorTrap` feels like the obvious answer | mirror beat gained too much contrast, symmetry, or isolation | keep the trap legible but colder; soften its highlight or interrupt its clean silhouette |
| `A302_Shot_05_WindowFalseLead` and `A302_Shot_06_PawnFalseLead` feel like one zone | false leads collapsed into one readable corner | separate props so the window and pawn beats resolve as different suspicions |
| `A302_Shot_07_EntryToAnswerLine` breaks | clutter blocks the clue-family-to-answer read | lower, move, or simplify the blocking prop before changing anything else |
| nursery clue cluster feels muddy on approach | too many props share the same contrast and height near the crib | reduce nearby clutter or restore one clear halo/crown/monitor read at a time |

## Recommended order of fixes

1. fix entry-first read failures
2. restore clue-to-answer linkage
3. reduce false-lead overreach
4. clean up approach-distance clutter

This order protects the main solve path before polishing the quieter decoys.

## Fast next-pass prompts

Reuse one of these lines in review notes when the issue is obvious:

- `next pass: reduce mirror dominance from entry`
- `next pass: restore nursery-to-sigil linkage`
- `next pass: separate window and pawn false leads`
- `next pass: clear the entry-to-answer sightline`
- `next pass: simplify crib-side clutter`
