# WorldShell Source-Art Notes

This folder contains surrounding-space and shell-threshold source art that frames entry into Apartment 302 and return out of it.
It is not runtime environment art, not Unreal content, and not implementation-ready level dressing.

Current files:

- `entry_threshold_signage_source.svg`
  - What it is: source-art sheet for the primary shell-to-case threshold sign language.
  - Intended carry-forward: use when later implementation needs a readable approach marker that frames case entry without over-selling the apartment as a haunted carnival.

- `return_path_guidance_source.svg`
  - What it is: source-art sheet for calm return-path and post-case routing guidance.
  - Intended carry-forward: use when later implementation needs exit-flow clarity after a case resolves or fails.

- `shell_corridor_threshold_dressing_source.svg`
  - What it is: source-art sheet for the surrounding corridor/threshold dressing family around Apartment 302.
  - Intended carry-forward: use to keep the shell-entry space procedural, readable, and slightly municipal rather than generic horror grime.

- `shell_entry_exit_wayfinding_set_source.svg`
  - What it is: compact wayfinding set for entry, active-case, return, and restricted-state signage.
  - Intended carry-forward: use as the reusable sign family for threshold and shell-return surfaces.

Boundary:

- This pack is source-art-only.
- Do not place these files into Unreal content folders.
- Do not treat these boards as final level layout proof.
- Keep the threshold language readable and slightly uncanny, but calmer than the apartment interior.
