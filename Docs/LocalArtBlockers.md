Use this file only for real art-production blockers.

Write each blocker in this format:

```text
time:
task:
blocked step:
what I tried:
visible error or conflict:
smallest decision needed from chat2:
```

Rules:

- do not write routine progress here
- do not use GitHub issues, PR comments, or pushes as the blocker channel
- after writing a blocker here, wait for `chat2` to respond through `Docs/LocalArtStatus.md`
- `chat2` remains the planning authority and milestone owner for new work
- `chat11` may flag stale states, unmanaged blockers, or documentation drift; treat those flags as coordination corrections, not as planner reassignment
- if the blocker implies code or Unreal integration work, stop and ask for reassignment instead of guessing
- if an active task cannot proceed, record the blocker here before going quiet
- if the task is not blocked but is waiting on a dependency or was reissued/canceled/parked, record that checkpoint in `Docs/LocalArtHandoff.md` instead of leaving the task silently stalled
